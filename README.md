Este repositório contém o frontend da plataforma CollabExercises, desenvolvido em React. A aplicação disponibiliza uma interface moderna e funcional para autenticação, navegação, gestão de exercícios e soluções. O frontend da plataforma CollabExercises foi desenvolvido para proporcionar uma experiência de utilizador simples, intuitiva e organizada, alinhada com os protótipos de design definidos.

Tecnologias Utilizadas

React

Vite

React Router

Axios

Instalação e Execução

1. Instalar dependências - npm install

2. Iniciar servidor de desenvolvimento - npm run dev

A aplicação ficará disponível em:

http://localhost:5173

A comunicação com o backend será realizada através de Axios.
A URL base prevista para desenvolvimento é:

http://localhost:5000
